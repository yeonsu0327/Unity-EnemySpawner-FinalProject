# 🎮 Unity Enemy Spawner Shooter Final Project

Unity 기말 프로젝트.
적(Enemy) 스폰, 플레이어 WASD 이동, 마우스 클릭 총알 발사, 간단한 UI 표시 기능을 포함한 기본 슈팅 게임입니다.

---

## 주요 기능

- ✅ **적 Spawner** : 일정 시간마다 7개의 적이 무작위로 생성됩니다.
- ✅ **플레이어 이동** : WASD 키로 이동 가능합니다.
- ✅ **총알 발사** : 마우스 클릭으로 총알 발사.
- ✅ **기본 UI** : 적 개수, 체력, 점수 등 상황 표시.
